# Plan 08-03: Trigger Chain Node Graph Editor

## Goal
The trigger editor gains a "Chain View" tab alongside the existing "Edit" tab. Chain View renders an imgui-node-editor canvas where:
- Each `TriggerEntry` = one node (showing ID, name, event count, action count)
- A "Trigger" action in trigger A that targets trigger B = a directed edge from A's output pin to B's input pin
- Clicking a node switches back to Edit tab and selects that trigger for editing
- User can pan/zoom the canvas freely

## Technology
Uses `ax::NodeEditor` (imgui-node-editor library at `F:\mmo\deps\imgui-node-editor`).  
Pattern: identical to `material_editor_instance.cpp` which already uses this library.

## Node/Pin/Link ID Scheme
To avoid ID collisions, allocate IDs deterministically from trigger IDs:
- Node ID = `trigger.id()` (uint32 fits in ax::NodeId)
- Input pin ID = `trigger.id() * 3 + 1`
- Output pin ID = `trigger.id() * 3 + 2` (one output pin; all outgoing chains share it)
- Link ID = hash of `(source_trigger_id << 16 | target_trigger_id)` — unique per pair

```cpp
void TriggerEditorWindow::DrawChainView(const proto::TriggerManager& triggers)
{
    // Lazy-create editor context
    if (!m_nodeEditorCtx)
    {
        ax::NodeEditor::Config config;
        config.SettingsFile = nullptr; // don't save layout to file
        m_nodeEditorCtx = ax::NodeEditor::CreateEditor(&config);
    }

    ax::NodeEditor::SetCurrentEditor(m_nodeEditorCtx);
    ax::NodeEditor::Begin("TriggerChain", ImVec2(0, 0));

    // Draw all trigger nodes
    for (int i = 0; i < triggers.triggers_size(); ++i)
    {
        const auto& trigger = triggers.triggers(i);
        const ax::NodeEditor::NodeId nodeId(trigger.id());

        ax::NodeEditor::BeginNode(nodeId);
        ImGui::Text("[%u] %s", trigger.id(), trigger.name().c_str());
        ImGui::Dummy(ImVec2(160, 0)); // min width

        // Stats row
        ImGui::TextDisabled("%d event(s)  %d action(s)",
            trigger.newevents_size(), trigger.actions_size());

        // Pins: input (left) and output (right)
        const ax::NodeEditor::PinId inputPin(trigger.id() * 3 + 1);
        const ax::NodeEditor::PinId outputPin(trigger.id() * 3 + 2);

        ax::NodeEditor::BeginPin(inputPin, ax::NodeEditor::PinKind::Input);
        ImGui::Text("▶");
        ax::NodeEditor::EndPin();

        ImGui::SameLine();
        ImGui::SetCursorPosX(ax::NodeEditor::GetNodeSize(nodeId).x - 24.0f);

        ax::NodeEditor::BeginPin(outputPin, ax::NodeEditor::PinKind::Output);
        ImGui::Text("▶");
        ax::NodeEditor::EndPin();

        ax::NodeEditor::EndNode();
    }

    // Draw links for all "Trigger" actions
    for (int i = 0; i < triggers.triggers_size(); ++i)
    {
        const auto& trigger = triggers.triggers(i);
        for (int j = 0; j < trigger.actions_size(); ++j)
        {
            const auto& action = trigger.actions(j);
            // trigger_actions::Trigger = action type 0 (check actual enum value)
            if (action.type() == trigger_actions::Trigger && action.has_target_name())
            {
                // target_name stores the target trigger ID as string for Trigger action
                // Check actual proto field — may be action.data() or action.target_trigger_id()
                uint32 targetId = 0;
                if (action.data_size() > 0)
                {
                    targetId = static_cast<uint32>(action.data(0));
                }
                if (targetId != 0)
                {
                    const uint32 linkId = (trigger.id() << 16) | (targetId & 0xFFFF);
                    ax::NodeEditor::Link(
                        ax::NodeEditor::LinkId(linkId),
                        ax::NodeEditor::PinId(trigger.id() * 3 + 2),    // source output
                        ax::NodeEditor::PinId(targetId * 3 + 1)          // target input
                    );
                }
            }
        }
    }

    // Handle node click → switch to Edit tab and select that trigger
    if (ax::NodeEditor::IsNodeSelected())
    {
        ax::NodeEditor::NodeId selectedNode;
        if (ax::NodeEditor::GetSelectedNodes(&selectedNode, 1) > 0)
        {
            m_jumpToTriggerId = static_cast<uint32>(selectedNode.Get());
        }
    }

    ax::NodeEditor::End();
    ax::NodeEditor::SetCurrentEditor(nullptr);
}
```
